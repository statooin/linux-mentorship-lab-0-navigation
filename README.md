# Linux Mentorship Lab 0: Navigation & File System Basics

This repository contains the first lab of the Linux Mentorship series, focusing on **essential navigation** and file system operations. Mastery of the command line interface (CLI) starts with the ability to move through the directory tree efficiently.

## 🎯 Objectives
The goal of this lab is to familiarize yourself with the Linux directory structure and the core tools used to manage files and folders. By the end of this lab, you will be able to:
* Navigate the Linux directory tree using absolute and relative paths.
* Create, copy, move, and delete files and directories.
* List file details and understand hidden files.
* Use wildcards and shortcuts for faster navigation.

## 📂 Linux File System Hierarchy
A key part of this lab is understanding the standard layout, including:
* `/` — The root directory.
* `/home` — User personal directories.
* `/etc` — System configuration files.
* `/bin` & `/sbin` — Essential binaries.
* `/tmp` — Temporary files.

## 📋 Core Commands Covered

| Command | Description |
| :--- | :--- |
| `pwd` | Print Working Directory (Where am I?) |
| `ls` | List directory contents (use `-la` for hidden files) |
| `cd` | Change directory (`cd ..`, `cd ~`, `cd -`) |
| `mkdir` | Create new directories |
| `touch` | Create empty files |
| `cp` | Copy files and directories (`cp -r` for folders) |
| `mv` | Move or rename files |
| `rm` | Remove files or directories (`rm -rf` with caution) |

## 🚀 Lab Tasks
1. **Explore**: Locate your current path and list all files, including hidden ones.
2. **Structure**: Create a nested directory tree (e.g., `project/src/data`) using a single command.
3. **Manipulate**: Practice moving files between directories and renaming them via CLI.
4. **Cleanup**: Safely remove the practice files and directories created during the lab.

## 💡 Pro Tips
* Use the **Tab** key for auto-completion.
* Use `cd -` to quickly toggle between your current and previous directory.
* Use `.` (current) and `..` (parent) to save time with relative paths.
